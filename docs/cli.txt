Definitions:
Data directories:
    someproject/data/
        raw/
        interim/
        processed/
        external/


Say John Smith has a dataset 'somedataset',
the project name would be: smith-somedataset.
Or if the dataset is called 'some-long-name-dataset-analysing-imdb'
call the project name: 'smith-imdb'.
The dataset 'somedataset' contains:
    somefile.csv
    somefiletwo.csv
    anotherfile.xlsx
    randomfile.docx

someproject = smith-somedataset


==================================================
dpx init someproject
==================================================
Creates:
    'someproject' project directory: projects/someproject/
    data directories inside this project directory: projects/someproject/data/raw, ...
    someproject.ipynb
    sources.txt
    README.md

NB: there is no data to make a db on


==================================================
dpx ls
dpx ls -p
==================================================

==================================================
dpx rm someproject anotherproject
dpx rm some project anotherproject -p
==================================================

==================================================
dpx copy-files -n someproject
==================================================
Copies files from someproject/data/raw to someproject/data/interim
renaming files from 'somefile.ext' to 'somefile-copy.ext


==================================================
dpx create-db -n someproject 
dpx create-db -n someproject --files somefile.csv somefiletwo.csv anotherfile.xlsx 
==================================================
Creates a sqlite db to someproject/data/external of data in someproject/data/raw


==================================================
dpx download kaggle --url https://www.kaggle.com/datasets/johnsmith/somedataset/data -n smith-somedataset
==================================================
Downloads the kaggle dataset using the api to project folder smith-somedataset


==================================================
dpx init-kaggle --url https://www.kaggle.com/datasets/johnsmith/somedataset/data -n smith-somedataset
==================================================
Calls:
    PROJNAME='smith-somedataset'
    KAGGLEURL='https://www.kaggle.com/datasets/johnsmith/somedataset/data'

    dpx init -n $PROJNAME
    dpx download kaggle --url $KAGGLEURL -n $PROJNAME 
    dpx copy-files -n $PROJNAME
    dpx create-db -n $PROJNAME
